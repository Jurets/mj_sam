# Description
The admin report plugin "Assesments" able to: 
* Download all assignments for selected student into one zipfile, including resubmissions
* Download all quiz results for selected student for all courses into one PDF

# Installation

## Install from zip
* Logout and login again as admin
* Unpack zip file to /<path_to_moodle>/report/
* Navigate on your moodle page to "Site administration->Notifivations"
* Follow the instructions 

## Install via Plugin installer
* Login as admin. Go to "Site administration->Plugins->Install plugins"
* Choose "Site report (report)" type
* Choose zip file "moodle_assesment_xxx.zip" (save it as e.g. "assesment.zip", set author as you wish)
* Click Install button
* After passed validation click "Install plugin"

# Usage
To run report:
* Login under admin role
* Click menu item "Site administration->Reports->Student assesments"
* Select student and click one of two buttons in row.
Enjoy!
